# 📸 Photography Portfolio Website

This project is a custom-built photography portfolio designed to showcase professional digital and film portrait work. Developed using semantic HTML5, modern CSS, and vanilla JavaScript, it emphasizes a minimalist aesthetic and intentional white space.

## 🌟 Features

- Responsive layout with clean, modern design
- Masonry-style grid using Masonry.js for dynamic image arrangement
- Lightbox modal with keyboard navigation for full-size image viewing
- Semantic structure and accessible navigation
- Social media and contact integration
- Fully mobile-ready and cross-browser tested

## 🛠️ Tech Stack

- HTML5
- CSS3 (with custom media queries for responsiveness)
- JavaScript (modular and readable)
- [Masonry.js](https://masonry.desandro.com/) for image layout
- Git & GitHub for version control and collaboration
